Projeto de scanner de portas desenvolvido em Python utilizando a biblioteca python-nmap, que atua como uma interface para o Nmap. O script realiza a varredura de um host alvo, identificando portas abertas, serviços em execução, versões dos serviços e informações adicionais, sendo útil para estudos de redes e segurança da informação.

O scanner utiliza as opções -sV (detecção de versão dos serviços) e -sC (execução de scripts padrão do Nmap), exibindo de forma organizada o estado das portas, protocolo, serviço, produto, versão e informações extras encontradas durante o scan.

Este projeto tem como objetivo o aprendizado prático de conceitos fundamentais de reconhecimento (recon), integração do Python com ferramentas de segurança e automação de análises básicas em ambientes controlados.

⚠️ Aviso: Este código foi desenvolvido exclusivamente para fins educacionais. A execução de scans deve ser feita apenas em sistemas próprios ou com autorização explícita. O uso indevido pode ser ilegal.
